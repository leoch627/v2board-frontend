### 🚀 使用指南

**快速开始**
```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env，设置 VITE_API_BASE_URL

# 3. 启动开发服务器
npm run dev
# 访问 http://localhost:3000

# 4. 生产构建
npm run build

# 5. 预览构建结果
npm run preview
```
